## 🎯 学习路径

### Level 1: 现代C++基础
1. 阅读 `ModernCPP.md` - 掌握现代C++特性
2. 理解智能指针、移动语义等核心概念

### Level 2: 性能优化入门
1. 学习 `Prefetch.md` - CPU预取技术
2. 阅读 `数值计算.md` - 数值计算优化

### Level 3: 底层原理深入
1. **必读**: `cpumemory.pdf` - 理解内存系统
2. 掌握缓存优化和NUMA架构

### Level 4: SIMD与向量化
1. 学习 `simd.pdf` - SIMD编程
2. 实践向量化优化技术

### Level 5: 极致性能
1. 研读 `optimizing_cpp.pdf` - C++优化大全
2. 深入 `optimizing_assembly.pdf` - 汇编级优化
3. 探索 `性能优化/` 目录的实战案例

## 💡 核心概念

- **缓存优化**: 利用CPU缓存层次提升性能
- **SIMD**: 单指令多数据并行
- **预取**: 提前加载数据到缓存
- **内存对齐**: 优化内存访问模式
- **RAII**: 资源获取即初始化
- **零成本抽象**: 现代C++的性能保证

## 📊 性能优化工具

- **perf**: Linux性能分析
- **VTune**: Intel性能分析器
- **Cachegrind**: 缓存分析工具
- **Compiler Explorer**: 查看汇编输出
